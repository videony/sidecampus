A propos

SideCampus est un logiciel opensource disponible sur Github. Créé en 2014, et rendu disponible à la communauté en 2018. Vous pouvez trouver le code source à cette URL: https://github.com/videony/sidecampus.

Ce site est une instance utilisant ce code source, mais non administré par les auteurs. Veuillez prendre contact par la page de contact pour autre information.
